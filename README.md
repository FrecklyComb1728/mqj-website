# mqj-website

基于 Nuxt（Minimal Starter）构建的站点项目。

## 环境要求

- Node.js > 20
- pnpm > 10.0.0

## 安装依赖

```bash
pnpm install
```

## 本地开发

启动开发服务器（默认：`http://localhost:3000`）：

```bash
pnpm dev
```

## 生产构建

构建生产包：

```bash
pnpm build
```

本地预览生产包：

```bash
pnpm preview
```
